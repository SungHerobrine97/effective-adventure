# The Return: 9012整合包文档

## 这是什么？

本包主要把2019年玩剩下的mod简单打包，并写了个挑战任务发布了出去。

本包没有匠魂和星系mod，他们两个不适配1.21.1版本（希望能引入吧），目前匠魂据说是要适配1.21.1的NeoForge。

## 安装

从[BBSMC网站](https://bbsmc.net/modpack/the-return-9012)下载文件，并使用PCL2/HMCL启动器导入并下载它（**你可能需要事先登录Minecraft，某些启动器不允许未登录Minecraft时下载内容**）。

如果遇到无法安装错误，请尝试将名字改为其他不带特殊字符的名称（**避免使用中文**）。

> 本整合包不适用于CurseForge和Modrinth客户端，将来会尝试发布。

## 设备要求

**RAM分配**：建议2GB RAM  
**OpenGL**：3.2

{% hint style="warning" %}

### 警告

请勿尝试安装FTB Quests Optimizer MOD，会导致打开任务界面时出现闪退。

***如果您安装了ModernUI Mod，请确保你的显卡OpenGL版本为3.3或更高版本。***

{% endhint %}
